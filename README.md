# Python Practice Projects

A working portfolio of small Python programs I've built while strengthening my fundamentals. Each file is a focused exercise — short enough to read in one sitting, but each one demonstrates a specific concept (variables, control flow, functions, state management, basic game logic). Together they form the foundation I built on for my MSc dissertation project: a [Retrieval-Augmented Generation chatbot for bus services in Stoke-on-Trent](https://github.com/Divyamthakur2/stoke-bus-rag-chatbot).

## About

I'm Divyam Thakur, an MSc Data Science graduate (University of Wolverhampton, 2025) targeting Data Analyst and Junior Data Scientist roles in the UK. This repo is where I practice and consolidate Python fundamentals — the layer underneath the SQL, pandas, and visualisation work I do day-to-day.

## What's Inside

Projects are grouped by the concept they practice rather than the order I built them, so it's easier to see what each one is actually demonstrating.

### Variables, data types, and formatted output

| File | What it demonstrates |
|---|---|
| `swap_glasses.py` | Swapping two variable values using a temporary variable |
| `Card.ipynb` | Storing structured personal data and producing formatted output |
| `math operators reference and practice.py` | Arithmetic operators, operator precedence, type casting between `int` and `float` |

### Conditional logic and user input

| File | What it demonstrates |
|---|---|
| `BMI_Calculator.py` | Reading numeric input, basic arithmetic, returning a single value |
| `BMI Calculator with interpretations.py` | Extending the above with `if`/`elif`/`else` branches to classify a BMI result |
| `Tip_Calculator.py` | Multi-step arithmetic, rounding, formatted currency output |
| `Roller coaster ride.ipynb` | Nested conditionals validating multiple eligibility rules (age, height) |
| `Python Pizza delivery.py` | A pricing engine that combines several conditional branches into a final total |

### State management and global variables

| File | What it demonstrates |
|---|---|
| `Bank Account Simulator.py` | Mutating program state across multiple operations using global variables |
| `Score Tracker.py` | Tracking running state across rounds; separating data from display logic |

### String manipulation and randomness

| File | What it demonstrates |
|---|---|
| `Band_Name_Generator.py` | String concatenation, basic user input, light text manipulation |
| `Traveller Budget Planner.py` | Combining variables and arithmetic to produce a structured trip estimate |

### Control flow and branching games

| File | What it demonstrates |
|---|---|
| `Treasure Island.py` | A choose-your-own-adventure game built entirely from nested conditionals — practice with branching logic and user-driven flow |

## Tech

- **Language:** Python 3
- **Editor:** PyCharm, Jupyter Notebook
- **Version control:** Git & GitHub

## Running Any Project

```bash
git clone https://github.com/Divyamthakur2/python-fundamentals.git
cd python-fundamentals
python "BMI Calculator with interpretations.py"
```

For the `.ipynb` notebooks, open them in Jupyter, JupyterLab, or VS Code.
